rangeslider.js

Simple, small and fast JavaScript/jQuery polyfill for the HTML5 <input type="range"> slider element.

Download rangeslider.js
Currently v{{ site.current_version }}

This project is on GitHub.

Features

  • Touchscreen friendly
  • Recalculates onresize so suitable for use within responsive designs
  • Small and fast

API

The rangeslider.js API is compatible with the standard HTML input methods.

{% highlight js %} $('input[type="range"]').val(10).change(); {% endhighlight %}

Simply calling something like this will just work.

Browser Support

Tested in IE8+ and all modern Browsers.

Installation

The easiest way to install the files is with Bower.

bower install --save rangeslider.js

You can also do it the old school way and download a zip file.

Usage

Using rangeslider.js is simple. Configuration over attributes.

{% highlight html %} {% endhighlight %}

Options

{% highlight js %} $('input[type="range"]').rangeslider({ // Feature detection polyfill: true, // CSS classes rangeClass: 'rangeslider', fillClass: 'rangeslider__fill', handleClass: 'rangeslider__handle', // Callback function onInit: function() {}, // Callback function onSlide: function(position, value) {}, // Callback function onSlideEnd: function(position, value) {} }); {% endhighlight %}
